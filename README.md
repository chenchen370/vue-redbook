# Vue Redbook

一个基于 Vue.js+express 的小红书模仿，包含评论，导航，主页，缓加载。

## 目录
- mysql数据库文件  用户密码改为你的 一般是 root 123456
- 在express-clientt目录下 cmd命令:node app.js
- 在my-project目录下 cmd命令:npm run serve
